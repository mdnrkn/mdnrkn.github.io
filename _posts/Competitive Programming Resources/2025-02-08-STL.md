`is_sorted()` : used to check whether the element of the given range is sorted or not in ascending order.

```cpp
vector<int> v = {1, 2, 3, 4, 5};
if (is_sorted(v.begin(), v.end())) cout << "Sorted";
else cout << "Not Sorted";
```
